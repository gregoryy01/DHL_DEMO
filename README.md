# WAYBILL_DHL

A Python desktop application for **waybill reading and logging** using a live camera feed or image input.

This project is designed for:
- webcam testing
- external/industrial camera demo integration
- barcode reading
- OCR-based waybill number extraction
- saving captured images and results into CSV for later use

## Features

- Live camera preview with Tkinter UI
- Camera scan, start, and stop controls
- Trigger-based capture
- Full-frame image saving
- Waybill number extraction
- Barcode reading support
- OCR text display
- CSV logging for later review
- Modular folder structure for future expansion

## Requirements

Python 3.10 or newer

Windows OS recommended for the provided .bat scripts

Webcam or external camera

Internet connection during initial package installation
